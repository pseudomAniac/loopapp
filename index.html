Application Brief

This reporting application was developed to display data taken directly from the main PNG LOOP Website. Right now it is limited to only showing the Reporter Story count. It the second updated version it will have the capabilities to show the count for stringers also.

To use this app, simply select a report from the main menu. There are 2014 Reports and 2015 Reports which can be located as sub menu's
